# The Event Project

* Ruby version : ruby '2.5.1'

* Rails version : rails '5.2.3'

* Configuration : ```git clone```

* Database creation : 
 ```rails db:create```
 ```rails db:migrate```
 ```rails db:seed```

* Run the server : ```rails s```

Local version : http://localhost:3000/

Heroku version : http://theeventproject.herokuapp.com/
